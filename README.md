### ⚙️ Technical Architecture
*   **State Management**: `flutter_bloc` for robust, event-driven reactive logic.
*   **Mapping**: `google_maps_flutter` integration (Configured for Android/iOS).
*   **Asset Management**: Custom `SmartImage` implementation for resilient network/local image handling and fallback states.
*   **Theming**: Centralized `AppTheme` with glassmorphic tokens and premium color palettes.
